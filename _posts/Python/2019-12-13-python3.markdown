### 系统函数
- ord('d') : 获取字符的ASCII
- input() : 获取用户输入
- print() : 打印数据
- round(a,2) : 四舍五入保留小数


python中递归有最大深度限制，可以调用系统库修改此配置

```text
import sys
sys.setrecursionlimit(10000)
```
